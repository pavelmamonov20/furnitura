# Улучшения качества штрих-кодов и их центровки

## Основные улучшения:

### 1. Повышение качества изображений:
- Увеличена высота модуля до 25.0 (было 15.0)
- Увеличена ширина модуля до 0.7 (было 0.45)
- Установлено разрешение DPI до 600 (было 300)
- Увеличены поля (quiet_zone) до 1.5 (было 1.0)

### 2. Центровка штрих-кодов:
- Штрих-коды теперь центрируются в изображении с отступами
- Штрих-коды центрируются в ячейках Excel
- Увеличены отступы: 30 пикселей сверху/снизу, 40 пикселей слева/справа

### 3. Улучшенные параметры:
- Используется алгоритм LANCZOS для масштабирования изображений
- Увеличена высота строк в Excel до 90 пунктов
- Расширена колонка C до 60 символов для лучшей центровки

### 4. Результаты:
- Штрих-коды стали более четкими и не размытыми
- Штрих-коды идеально центрированы в ячейках Excel
- Улучшена сканируемость штрих-кодов
- Обеспечена совместимость с различными сканерами

## Файлы:
- `centered_quality_barcodes.py` - основной скрипт с улучшениями
- `тест_штрихкоды_с_улучшенным_качеством.xlsx` - тестовый файл с результатами
