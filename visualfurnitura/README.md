# VisualFurnitura

VisualFurnitura - приложение для визуализации установки фурнитуры на оконные и дверные конструкции.

## Описание

VisualFurnitura - это приложение для Windows, разработанное на Python 3.11+ с использованием PyQt6, которое позволяет:
- Визуализировать схемы установки фурнитуры на окна
- Парсить PDF-файлы с заданиями и извлекать информацию о фурнитуре
- Управлять базой данных компонентов и профильных систем
- Экспортировать схемы и перечни компонентов в PDF

## Требования

- Python 3.11 или выше
- Windows OS

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск приложения

```bash
python main.py
```

## Архитектура приложения

- `db_manager.py` - управление SQLite базой данных
- `pdf_processor.py` - парсинг PDF файлов
- `visualization/` - модули визуализации (2D отрисовка)
- `admin/` - модули администрирования
- `pdf_exporter.py` - генерация PDF отчетов
- `hardware_calculator.py` - расчет координат установки

## Функциональность

### Для администратора:
- Добавление/редактирование компонентов фурнитуры
- Управление профильными системами
- Загрузка изображений компонентов
- Настройка точек крепления

### Для монтажников:
- Загрузка PDF-файлов с заданиями
- Автоматическое распознавание артикулов и количества компонентов
- 2D визуализация схемы окна с фурнитурой
- Масштабирование и перемещение схемы
- Просмотр информации о компонентах
- Экспорт в PDF

## Технологии

- **Python 3.11+**
- **PyQt6** - GUI и 2D визуализация
- **SQLite3** - встроенная база данных
- **PyMuPDF (fitz)** - парсинг PDF
- **ReportLab** - генерация PDF
- **Pillow** - работа с изображениями
- **NumPy** - математические расчеты

## Структура базы данных

База данных включает таблицы:
- `hardware_components` - компоненты фурнитуры
- `profile_systems` - профильные системы
- `orders` - заказы/проекты
- `order_hardware` - назначение фурнитуры заказам
